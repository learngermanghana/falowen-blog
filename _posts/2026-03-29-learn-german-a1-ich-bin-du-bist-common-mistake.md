If you are learning German A1, this is one of the first grammar points you need:

- **ich bin** = I am  
- **du bist** = you are (informal)

Many beginners mix them.
This causes basic speaking errors in introductions and daily conversation.

In this lesson, you will learn the correct pattern and practice it with objective questions.

## The verb: **sein** (to be)
In German, **sein** changes based on the subject.

At A1, the two forms you must master first are:

- **ich bin**
- **du bist**

### Correct examples
- *Ich bin Maria.*
- *Ich bin müde.*
- *Du bist mein Freund.*
- *Du bist heute spät.*

---

## DON’T make this mistake
A very common error:

✗ *Ich bist Student.*  
✓ *Ich bin Student.*

Another common error:

✗ *Du bin neu hier.*  
✓ *Du bist neu hier.*

### Quick memory trick
- **ich → bin**
- **du → bist**

Say it out loud 5 times:

> Ich bin. Du bist. Ich bin. Du bist.

This helps your mouth and ear learn the pattern together.

---

## Mini dialogue (A1)
**A:** Hallo! Ich bin Daniel.  
**B:** Hi Daniel, ich bin Sara.  
**A:** Bist du neu hier?  
**B:** Ja, ich bin neu.

Use this model when introducing yourself.

## Objective practice (with answers)
Choose the best option for each question.

### 1) ____ Maria.
- [ ] Ich bist  
- [x] Ich bin  
- [ ] Du bin  
- [ ] Du bist  

**Answer:** **Ich bin** Maria.

### 2) ____ mein Freund.
- [ ] Ich bist  
- [ ] Ich bin  
- [ ] Du bin  
- [x] Du bist  

**Answer:** **Du bist** mein Freund.

### 3) Which sentence is correct?
- [ ] Ich bist neu hier.  
- [ ] Du bin neu hier.  
- [x] Ich bin neu hier.  
- [ ] Du bist aus Ghana. *(Correct grammar, but not the intended “I am new here” sentence.)*

**Answer:** **Ich bin neu hier.**

### 4) Fix this mistake: *Du bin sehr freundlich.*
- [ ] Ich bin sehr freundlich.  
- [x] Du bist sehr freundlich.  
- [ ] Du bin sehr freundlich.  
- [ ] Ich bist sehr freundlich.  

**Answer:** **Du bist sehr freundlich.**

### 5) Which memory rule is correct?
- [x] ich → bin, du → bist  
- [ ] ich → bist, du → bin  
- [ ] ich → sein, du → bin  
- [ ] ich → du, du → ich  

**Answer:** **ich → bin, du → bist**

---

## Final tip
Do not rush to long grammar tables first.
Master small patterns like **ich bin / du bist** until they are automatic.
That is how A1 speaking becomes confident and correct.
